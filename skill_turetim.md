# 📚 Türkiye Türkçesinin Türetim Ekleri — Kapsamlı Bilgi Dosyası

> **Kaynak:** Uzun, N.E., Uzun, L.S., Aksan, Y.K. & Aksan, M. (1992). *Türkiye Türkçesinin Türetim Ekleri — Bir Döküm Denemesi*. Ankara: Şirin Kırtasiye.
> **Yöntem:** TDK Türkçe Sözlük 7. Baskı veritabanı taranmış, 14.635 türetimsel maddebaşından 191 türetim eki saptanmıştır.
> **Yaklaşım:** Eşzamanlı (synchronic), veritabanlı — tüm artzamanlı (diachronic) bilgiler dışlanmıştır.

## 1. Temel Kavramlar ve Terminoloji

### 1.1 Taban Türleri (Base Types)

| Terim | Açıklama | UD Karşılığı |
|-------|----------|--------------|
| **Adsıl (Nominal)** | İsim, sıfat, belirteç, zamir taban | NOUN, ADJ, ADV, PRON |
| **Eylemcil (Verbal)** | Fiil taban | VERB |
| **Ad (İsim)** | Salt isim | NOUN |
| **Sıfat** | Salt sıfat | ADJ |
| **Belirteç** | Zarf | ADV |
| **Ad.Sıfat** | Hem isim hem sıfat kullanımı olan | NOUN/ADJ |
| **Sıfat.Belirteç** | Hem sıfat hem belirteç | ADJ/ADV |
| **Belirteç.Ad** | Hem belirteç hem isim | ADV/NOUN |
| **Adıl** | Zamir | PRON |
| **İlgeç** | Edat/postpozisyon | ADP |
| **Bağlaç** | Bağlaç | CCONJ/SCONJ |

### 1.2 Türetim Türleri (Derivation Types)

| Kısaltma | Taban → Türev | Açıklama |
|----------|---------------|----------|
| **A→A** | Adsıl → Adsıl | İsimden isim (küçültme, meslek, soyutluk...) |
| **A→E** | Adsıl → Eylemcil | İsimden fiil (başla-, güzelleş-) |
| **E→A** | Eylemcil → Adsıl | Fiilden isim (yaz-ı, dol-um) |
| **E→E** | Eylemcil → Eylemcil | Fiilden fiil (ettirgen, edilgen, işteş) |

### 1.3 Diğer Sınıflandırma Boyutları

| Boyut | Değerler |
|-------|----------|
| **Taban yapısı** | Yalın, Bileşik, Yansıma |
| **Köken** | Yerli, Yabancı (alıntı) |
| **Yaş** | Eski (yerleşik), Yeni (son dönem üretim) |

### 1.4 İÇİ — İçerikle Çözümleme İlişkisi

Kitabın temel yöntemi: Bir sözcüğün türetimsel olarak çözümlenebilmesi için taban ve türev arasında **sözlüksel içerik ilişkisi** (İÇİ) kurulabilmesi gerekir. Taban, sözlükte yalın maddebaşı olarak bulunmalıdır. Bu kısıtlama, sezgisel çözümlemeyi dışlayarak tutarlı bir çerçeve sağlar.

## 7. Morfolojik Çözümleme İçin Kurallar

### 7.1 Taban Belirleme (Kitabın İlkeleri)

1. **İÇİ zorunluluğu**: Taban ve türev arasında sözlüksel içerik ilişkisi kurulabilmeli
2. **Sözlük doğrulaması**: Taban, sözlükte yalın maddebaşı olarak bulunmalı
3. **İkili kök sorunu**: göç/göç-, ekşi/ekş- gibi hem adsıl hem eylemcil kök durumlarında, İÇİ ancak birinin çözümlenişini olanaklı kılıyorsa o tercih edilir
4. **Bileşik ek**: Taban-ek ikilisi başka maddebaşlarında bulunmuyorsa, parçaların tümü tek birleşik ek sayılır
5. **Koruyucu ünsüzler**: (y), (n), (s), (ş) — taban ünlüyle bittiğinde eklenir

### 7.2 Ek Sıralama Kuralları (Çatı Sırası)

Kitaptaki verilere göre gözlenen sıralama:

```
[KÖK] + [DÖNÜŞLÜ/EDİLGEN] + [İŞTEŞ] + [ETTİRGEN] + [OLUMSUZ] + [çekim ekleri]
```

Zincirleme türetim örnekleri:
- yaz + dır + ıl + abil + ecek → yazdırılabilecek (E→E→E→çekim)
- güzel + leş + tir → güzelleştir (A→E→E)
- ev + len + dir + il → evlendiril (A→E→E→E)

### 7.3 Allomorf Seçimi

| Ek | Ortam | Allomorf |
|----|-------|----------|
| -(D)Ir- | Tek heceli fiil + ünlü sonu | -(I)r- (kop-ar-) |
| -(D)Ir- | Çok heceli fiil | -(I)t- (ürk-üt-) |
| -(I)l- | Ünsüzle biten gövde | -(I)l- (yaz-ıl-) |
| -(I)n- | Ünlüyle biten gövde | -(I)n- (oyna-n-) |
| -(l)A- | Ünlüyle biten gövde | -lA- (taş-la-) |
| -(l)A- | Ünsüzle biten gövde | -A- (baş-a-? → başla-) |
